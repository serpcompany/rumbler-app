# Feature: Venue Bookings

**Flow**
1) From chat → Book → pick verified gym & time
2) Choose ruleset (Boxing/Muay Thai/BJJ/MMA), gear checklist
3) Optional deposit/fee (Stripe - real-world service)
4) Check-in allowed only within ~100m geofence

**Non-Goals (MVP)**
- Gym admin portal (later)
- Automated payouts (later)
