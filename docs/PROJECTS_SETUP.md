# GitHub Projects Setup

1) Create Project: `Rumbler Roadmap` linked to repo.
2) Add fields: Status (Backlog, Planned, In Progress, In Review, Done), Priority (P1/P2/P3), Type (feature/bug/design), Milestone, Target Date.
3) Views: Board, Table, Timeline.
4) Labels: type:feature, type:bug, type:design, mvp, post-mvp, frontend, backend, mobile.
5) Milestones: MVP Design, MVP API, MVP Mobile, Beta Launch.
